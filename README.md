# Inventory Discrepancy Analysis

## Project Overview

This project focuses on inventory discrepancy analysis for post-sales supply chain operations. The objective is to identify differences between inventory records across multiple systems, including warehouses, repair centers, and internal inventory management platforms.

The analysis helps improve inventory visibility, data accuracy, operational efficiency, and decision-making by detecting shortages, excess inventory, and mismatched stock records.

---

## Business Problem

In global post-sales operations, inventory discrepancies can impact:

* Spare parts availability
* Forecast accuracy
* Warranty support operations
* Customer service levels
* Supply chain continuity

This project was created to automate discrepancy detection and provide actionable insights through data analysis.
